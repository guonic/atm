# 12%筹码集中度选股策略指南

## 概述

12%筹码集中度选股策略是一个专为散户投资者设计的"精准筛选"系统。该策略通过分析筹码集中度和吸筹形态，识别主力已完成充分吸筹的优质标的，从源头降低选股风险，避免垃圾股陷阱。

**核心哲学**：筹码是股价上涨的根本驱动力。掌握12%这个黄金阈值，配合相应的吸筹形态，选股变得简单，稳定盈利触手可及。

## 策略原理

### 一、为什么散户总是选错股票？

#### 1. 常见困境

**问题**：
- 持有弱势股长期横盘，错过市场上涨
- 追涨看似强势的股票，却遇到主力出货，深度套牢

**根本原因**：
- 选股不是追热点、追题材
- 而是要理解"主力意图"隐藏在"筹码分布"中

#### 2. 12%筹码集中度的核心价值

**定义**：
- 筹码集中度 = (90%持有者成本区间上限 - 90%持有者成本区间下限) / (上限 + 下限)
- 数值越低，筹码越集中，主力控盘能力越强
- 数值越高，筹码越分散，股价多由散户主导，波动无序

**12%阈值**：
- 这是经过万次实盘验证的"控盘阈值"
- 当筹码集中度 ≤ 12% 时，90%的筹码聚集在狭窄价格区间
- 市场浮动筹码极少，说明主力已完成充分吸筹，具备强势控盘能力
- 当集中度 > 12% 时，筹码分散在宽泛区间，要么主力未完成吸筹，要么已开始出货，风险极高

**策略价值**：
- 用12%这个明确阈值快速区分主力控盘股与散户股
- 从源头降低选股风险

### 二、两种核心吸筹形态

单纯的12%集中度不足以判定优质标的，需搭配主力吸筹的典型筹码形态形成共振。

#### 形态一：低位温和吸筹（低位横盘+12%集中度）

**特征**：
- 长期横盘消磨散户耐心，逼其在低位交出筹码
- 尤其在牛市初期，当其他标的纷纷上涨时，这类横盘标的更易让散户跟风换股
- 形成低位密集筹码区，其他价位筹码基本消失

**量化标准**：
- 横盘时长 ≥ 20个交易日
- 90%筹码成本区间集中度 ≤ 12%
- 横盘阶段成交量较前期下跌阶段均值萎缩30%以上

#### 形态二：打压至低平台吸筹（二次横盘+12%集中度）

**特征**：
- 当长期横盘无法逼出坚定持股或深度套牢的散户时，主力采用"打压+二次横盘"的吸筹手法
- 彻底击溃散户持股信念，上涨动力更强

**三阶段判断**：
1. **第一阶段**：首次低位横盘时，形成密集筹码区，但上方仍有另一处密集成交区
2. **第二阶段**：股价被打压至更低平台横盘，上方密集筹码逐步消失
3. **第三阶段**：新平台形成密集筹码区，90%筹码成本区间集中度 ≤ 12%，且筹码分布图仅留存两个平台的密集区

**量化标准**：
- 两次横盘总时长 ≥ 30个交易日
- 90%筹码成本区间集中度 ≤ 12%
- 横盘阶段成交量较前期下跌阶段均值萎缩30%以上

## 买入信号：4步量化落地

### 第一步：阈值筛选，建立初选池

**筛选条件**：
- 筹码集中度 ≤ 12%

**排除条件**（严格排除）：
1. 股价处于高位区域，前期累计涨幅超过50%
2. 近期频繁放量震荡，但未形成稳定密集筹码区

**输出**：初选池8-10只股票

### 第二步：形态验证，精简观察池

**验证条件**（确认目标满足两种形态之一）：
- 形态一：低位温和吸筹
- 形态二：打压至低平台吸筹

**关键观察**：
- 是否形成稳定的低位密集筹码区
- 其他价位筹码是否已清空
- 形态二需确认两次横盘的筹码转换完成，上方旧密集区消失

**输出**：观察池5-6只股票

### 第三步：确认换手，锁定介入标的

**核心验证 - 横盘时长**：
- 形态一：横盘时长 ≥ 20个交易日
- 形态二：两次横盘总时长 ≥ 30个交易日

**成交量验证**：
- 横盘阶段成交量较前期下跌阶段均值萎缩30%以上
- 表明抛压耗尽，筹码锁定良好

**输出**：可介入标的

### 第四步：精准介入，把控节奏与仓位

**介入时机**：
- 股价突破低位密集筹码区当日
- 成交量较前5日均值放大50%以上（量价齐升确认突破有效性）

**分仓策略**：
- **首次试仓**：用30%资金试仓
- **加仓**：突破后3个交易日内股价未回踩突破位，再加仓40%
- **预留资金**：预留30%机动资金

## 卖出信号：动态调整

### 持有条件

**坚定持有**（满足以下条件）：
- 股价突破后，量能持续在线
- 集中度维持在 ≤ 12%

### 离场条件（满足任一条件）

1. **量能萎缩**：成交量显著萎缩，表明资金流出
2. **股价回落**：股价回落至突破位以下，表明突破失败
3. **集中度上升**：筹码集中度 > 12%，表明筹码开始分散，主力可能出货

## 风险控制

### 止损

- **固定百分比止损**：买入后，如果股价从买入价下跌超过5%，立即止损

### 止盈

- **固定百分比止盈**：买入后，如果股价从买入价上涨超过15%，立即止盈

### 仓位管理

- **分批建仓**：30%初始 + 40%加仓 + 30%预留
- **降低风险**：避免一次性满仓，降低单次交易风险

## 参数说明

### 筹码集中度参数

- `concentration_threshold`（默认：0.12）：筹码集中度阈值（12%）
  - 用于判断主力控盘程度
  - 数值越小，要求越严格

### 吸筹形态参数

- `pattern1_min_days`（默认：20）：形态一最小横盘天数
  - 用于验证低位温和吸筹的充分性
  - 数值越大，要求越严格

- `pattern2_min_days`（默认：30）：形态二最小横盘天数
  - 用于验证打压至低平台吸筹的充分性
  - 数值越大，要求越严格

### 成交量参数

- `volume_shrink_threshold`（默认：0.3）：横盘阶段成交量萎缩阈值（30%）
  - 用于验证抛压耗尽
  - 数值越大，要求越严格

- `volume_breakout_threshold`（默认：1.5）：突破时成交量放大阈值（50%增加 = 1.5倍）
  - 用于确认突破有效性
  - 数值越大，要求越严格

### 仓位管理参数

- `initial_position_pct`（默认：0.3）：初始仓位百分比（30%）
  - 用于控制首次试仓规模

- `add_position_pct`（默认：0.4）：加仓百分比（40%）
  - 用于控制加仓规模

- `add_position_days`（默认：3）：加仓等待天数
  - 用于确认突破有效性后再加仓

### 风险控制参数

- `stop_loss_pct`（默认：0.05）：止损百分比（5%）
  - 用于限制单笔交易最大亏损

- `take_profit_pct`（默认：0.15）：止盈百分比（15%）
  - 用于锁定利润

## 快速开始

### 1. 运行回测

```bash
python python/tools/analysis/evaluate_chip_concentration_strategy.py \
  --start-date 2024-01-01 \
  --end-date 2025-12-01 \
  --num-stocks 200 \
  --skip-market-cap-filter \
  --concentration-threshold 0.12 \
  --pattern1-min-days 20 \
  --pattern2-min-days 30 \
  --volume-shrink-threshold 0.3 \
  --volume-breakout-threshold 1.5 \
  --initial-position-pct 0.3 \
  --add-position-pct 0.4 \
  --add-position-days 3 \
  --stop-loss-pct 0.05 \
  --take-profit-pct 0.15 \
  --output outputs/chip_concentration_results.csv
```

### 2. 参数调整建议

**短线交易**：
- 集中度阈值：可适当放宽至0.15（15%）
- 横盘天数：可适当缩短（形态一15天，形态二25天）
- 成交量阈值：可适当放宽

**中线交易**：
- 集中度阈值：保持0.12（12%）
- 横盘天数：保持默认值
- 成交量阈值：保持默认值

**高波动品种**：
- 集中度阈值：可适当放宽至0.15
- 止损止盈：可适当放宽（止损7%，止盈20%）

**低波动品种**：
- 集中度阈值：可适当收紧至0.10
- 止损止盈：可适当收紧（止损3%，止盈10%）

## 策略优势

1. **精准筛选**：用12%阈值快速区分主力控盘股与散户股
2. **从源头降低风险**：避免垃圾股陷阱
3. **系统化操作**：4步量化落地，新手直接套用
4. **分批建仓**：降低单次交易风险
5. **风险控制**：内置止损止盈机制

## 风险提示

1. **筹码峰计算**：需要准确的筹码分布数据，当前使用占位符实现
2. **数据依赖**：需要历史成交量和价格数据
3. **计算复杂度**：实际筹码峰计算可能较复杂
4. **市场环境**：在极端市场环境下可能失效
5. **参数优化**：需要针对不同市场和标的进行优化

## 注意事项

1. **严格排除高位股**：前期累计涨幅超过50%的股票应严格排除
2. **避免频繁震荡**：近期频繁放量震荡但未形成稳定密集筹码区的股票应排除
3. **确认突破有效性**：必须等待量价齐升确认突破有效性后再介入
4. **分批建仓**：不要一次性满仓，预留机动资金
5. **及时止损**：一旦触发止损条件，立即离场，不要抱有侥幸心理

## 筹码峰指标实现

### 当前状态

策略使用 `DummyChipPeakIndicator` 作为占位符，它返回基于价格范围的模拟数据。

### 未来实现方向

实际筹码峰计算可以基于：

1. **历史成交量分布**
   - 统计不同价格区间的成交量
   - 计算筹码在不同价格区间的分布
   - 计算90%持有者成本区间的上下限

2. **持仓成本模拟**
   - 基于历史成交模拟持仓成本
   - 计算不同价格区间的持仓量
   - 识别密集筹码区

3. **第三方数据源**
   - 使用专业的筹码分布数据
   - 集成第三方筹码分析API
