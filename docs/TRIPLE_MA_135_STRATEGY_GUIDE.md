# 135均线交易策略指南

## 概述

135均线策略是一个专为散户投资者设计的"简单实用"趋势捕捉系统。该策略通过13日、34日、55日三根均线的组合，实现"资金动向 + 趋势判断 + 安全底线"的三重验证，帮助投资者精准找到大牛起涨点，避免追高和盲目抄底。

**核心哲学**：三根线各管一摊，凑齐了才是大牛信号。13线看资金活不活，34线看趋势拐不拐，55线看底线牢不牢。

## 策略原理

### 一、为什么散户总是错过大牛？

#### 1. 传统均线的盲点

**问题**：
- 90%的散户投资者亏损，主要原因在于均线选择不当
- 要么被短期均线误导，要么被长期均线拖慢
- 无法准确把握市场趋势

**常见错误**：

**5日均线**：
- 死守5日线的投资者容易被日线波动左右
- 价格稍微站上5日线就匆忙进场，第二天就被回调套牢
- 误把散户跟风的"假动能"当作真正的机会

**20日均线**：
- 用20日线抄底的投资者容易把反弹误判为反转
- 在弱市中，20日线偶尔向上，看似见底信号
- 但主力资金尚未完成建仓，短暂上涨两天后继续下跌，难以脱身

**60日均线**：
- 用60日线判断趋势的投资者容易错过最佳时机
- 60日线反应太慢，等它走平向上时，大牛已经涨了10%以上
- 只能吃到"残羹剩饭"，错过主升浪

#### 2. 135均线的优势

**解决方案**：
- 13日线：看短期资金动向，过滤70%假行情
- 34日线：定中期趋势方向，区分反弹和反转
- 55日线：守长期安全底线，主力不会轻易跌破

**核心逻辑**：
- 三根线配合好，大牛起涨信号藏不住
- 每根线的作用都至关重要，缺一不可

### 二、三根均线的核心职责

#### 1. 13日均线：看资金活不活

**职责**：
- 观察短期资金动向
- 过滤70%的假行情
- 作为短期资金的"晴雨表"

**特点**：
- 比5日线更稳定，不易被日线波动左右
- 比20日线更灵敏，能及时捕捉资金入场的痕迹

**判断标准**：
1. 13日线必须从走平位置逐渐向上
2. 价格必须连续3天站在均线上方1%以上
3. 单日向上不够，连续稳定才有效

**实战意义**：
- 连续3天站稳，说明资金真实流入
- 这是大牛启动前的"热身"信号

#### 2. 34日均线：看趋势拐不拐

**职责**：
- 定义中期趋势方向
- 区分反弹和反转
- 覆盖主力建仓周期（30-40天）

**特点**：
- 比20日线更长，能准确识别中期趋势
- 避免把反弹误判为反转

**判断标准**：
1. 34日线必须从向下转为走平，再缓慢向上
2. 13日线必须向上穿过34日线（金叉）
3. 金叉后，13日线不能回落至34日线下方

**核心逻辑**：
- 金叉就像推车上坡，13日线是"推力"，34日线是"坡度"
- 推力足够且坡度向上，价格才能稳定上涨
- 如果金叉后13日线回落，说明推力不足，只是短期反弹

#### 3. 55日均线：看底线牢不牢

**职责**：
- 作为长期安全底线
- 主力不会轻易跌破
- 是主力的"成本线"和投资者的"安全垫"

**特点**：
- 真正的牛市，价格绝对不会在启动前跌破55日线
- 如果跌破，主力自己也会被套，所以会坚决护盘

**判断标准**：
1. 34日线必须站在55日线上方
2. 55日线必须从向下转为走平，最好也向上
3. 价格绝对不能跌破55日线

**实战意义**：
- 55日线向上，说明长期趋势转暖
- 为上涨提供坚实基础

### 三、双重金叉共振

#### 1. 第一次金叉：13日线向上穿34日线

**条件**：
- 13日线向上穿过34日线
- 金叉后2天内，13日线要站在34线上方1%以上
- 不能金叉完就跌回去

**意义**：
- 确认短期资金活跃
- 短期趋势开始转强

#### 2. 第二次金叉：34日线向上穿55日线

**条件**：
- 34日线向上穿过55日线
- 金叉后2天内，34日线要站在55线上方1%以上
- 中期趋势才算真反转

**意义**：
- 确认中期趋势反转
- 长期趋势开始转暖

#### 3. 时间要求

**条件**：
- 两次金叉间隔不能超过5天
- 期间股价慢慢涨，单日涨1%-2%就行
- 别着急涨，急涨容易回调

**排除标准**：
- 金叉后立即被破坏的信号是假信号
- 两次金叉间隔超过10天的信号是假信号
- 说明主力无意真正拉升，应直接排除

## 买入信号：等待回调到位

### 第一步：确认双重金叉共振

**条件**：
1. 13日线向上穿过34日线（第一次金叉）
2. 34日线向上穿过55日线（第二次金叉）
3. 两次金叉间隔不超过5天
4. 金叉后均线保持稳定，不回落

**这一步的目的**：
- 确认趋势启动信号
- 过滤假信号

### 第二步：等待回调到关键位置

**回调位置**：
- 优先等回调到34日线附近，34线支撑最稳
- 如果34线撑得住，也能回调到13日线
- 但绝对不能跌破55日线，跌破就放弃

**量能要求**：
- 回调时成交量要慢慢缩
- 缩到之前金叉时平均量能的一半以下最好
- 别放量跌，放量跌说明资金在跑，不是真回调

**止跌信号**：
- 回调到均线附近后，收小阳线或十字星
- 第二天股价能回升到均线上方，说明跌不动了

### 第三步：潜伏时机

**入场条件**：
1. 双重金叉已完成
2. 价格回调到34日线附近（或13日线）
3. 价格保持在55日线上方
4. 成交量收缩
5. 出现止跌信号

**操作建议**：
- 止跌当天先轻仓买
- 第二天成交量比回调时多30%以上、股价不回落，再加仓

## 卖出信号：趋势破位

### 1. 价格跌破55日线

**条件**：
- 收盘价跌破55日线
- 当天收不回来

**意义**：
- 55日线是底线，跌破说明支撑失效
- 主力没有护盘，之前的信号可能是假的

**操作**：
- 不管亏多少都赶紧卖
- 别抱有侥幸心理

### 2. 13日线跌破34日线

**条件**：
- 13日线向下穿过34日线
- 说明短期趋势转弱

**意义**：
- 短期资金开始流出
- 趋势可能反转

**操作**：
- 及时离场，保护利润

### 3. 价格明显跌破34日线

**条件**：
- 价格跌破34日线超过容忍度（默认1%）
- 说明中期趋势可能转弱

**意义**：
- 中期支撑失效
- 需要及时止损

**操作**：
- 保护仓位，及时离场

## 四个避坑原则

### 1. 大盘55日线向下，坚决空仓

**原则**：
- 大盘长期弱，个股再强也难独善其身
- 主力拉升意愿低，信号大概率失效
- 不如等着

**操作**：
- 大盘55日线向下时，不操作个股
- 等待大盘企稳后再操作

### 2. 共振时量能骤减，直接排除

**原则**：
- 金叉时成交量突然变少，还收长上影线
- 说明资金犹豫，没信心推股价
- 后续难大涨

**操作**：
- 发现量能骤减，直接排除
- 不参与这种信号

### 3. 个股有基本面利空，绝不碰

**原则**：
- 操作前看近1个月公告
- 要是有连续亏损、监管问询、行业利空
- 再好看的均线也没用，主力可能会跑路

**操作**：
- 基本面有问题的股票，不操作
- 技术分析不能替代基本面分析

### 4. 跌破55日线，立即止损

**原则**：
- 55线是底线，跌破就说明主力没护盘
- 之前的信号可能是假的
- 再拿着只会亏更多

**操作**：
- 跌破55日线，立即止损
- 不要抱有侥幸心理

## 实战操作要点

### 1. 参数设置

**均线参数**：
- 短期：13日（推荐白色）
- 中期：34日（推荐黄色）
- 长期：55日（推荐紫色）

**重要提示**：
- 不要随意更改参数
- 这些参数是20年实战验证的
- 改参数会让信号不准确

**界面要求**：
- 三根线要清晰可见
- 13日线在上，34日线在中，55日线在下
- 形成"阶梯状"排列

### 2. 操作流程

**第一步：调整均线参数**
1. 打开股票软件
2. 设置13日、34日、55日均线
3. 确认三根线清晰可见

**第二步：验证双重金叉**
1. 观察13日线是否向上穿过34日线
2. 观察34日线是否向上穿过55日线
3. 确认两次金叉间隔不超过5天
4. 确认金叉后均线保持稳定

**第三步：等待回调**
1. 确认双重金叉已完成
2. 等待价格回调到34日线附近
3. 观察成交量是否收缩
4. 确认价格保持在55日线上方

**第四步：入场**
1. 出现止跌信号（小阳线或十字星）
2. 第二天成交量增加30%以上
3. 股价不回落，加仓

**第五步：离场**
1. 价格跌破55日线，立即止损
2. 13日线跌破34日线，及时离场
3. 价格明显跌破34日线，保护仓位

### 3. 常见陷阱

**陷阱一：单根均线信号**
- **问题**：仅凭一根均线就买入
- **后果**：容易被假信号误导
- **解决**：必须等待双重金叉共振

**陷阱二：参数不匹配**
- **问题**：随意更改均线参数
- **后果**：信号不准确
- **解决**：严格按照13/34/55设置

**陷阱三：忽视回调**
- **问题**：金叉后立即买入
- **后果**：追高被套
- **解决**：必须等待回调到位

**陷阱四：忽视基本面**
- **问题**：只看技术面，忽视基本面
- **后果**：踩到基本面利空
- **解决**：操作前检查基本面

## 参数说明

### 均线参数

- `ema_short`（默认：13）：短期均线周期
  - 用于观察短期资金动向
  - 数值越小，反应越敏感

- `ema_mid`（默认：34）：中期均线周期
  - 用于判断中期趋势方向
  - 覆盖主力建仓周期

- `ema_long`（默认：55）：长期均线周期
  - 用于守长期安全底线
  - 作为主力的成本线

### 金叉参数

- `cross_gap`（默认：5）：两次金叉的最大间隔天数
  - 用于确保双重金叉的时效性
  - 数值越大，要求越宽松

### 回调参数

- `pullback_tolerance`（默认：0.01）：价格接近34日线的容忍度
  - 用于判断价格是否回调到位
  - 数值越大，要求越宽松

### 2. 参数调整建议

**短线交易**：
- 均线参数：保持（13,34,55）
- 金叉间隔：可适当缩短至3天
- 回调容忍度：可适当放宽至1.5%

**中线交易**：
- 均线参数：保持（13,34,55）
- 金叉间隔：保持5天
- 回调容忍度：保持1%

**高波动品种**：
- 回调容忍度：可适当放宽至2%
- 更宽松的回调要求

**低波动品种**：
- 回调容忍度：可适当收紧至0.5%
- 更严格的回调要求

## 策略优势

1. **简单实用**：不需要复杂分析，按步骤操作即可
2. **三重验证**：资金动向 + 趋势判断 + 安全底线
3. **避免追高**：等待回调到位再买入
4. **避免抄底**：双重金叉确认趋势反转
5. **系统化操作**：从"凭感觉"变成"有章可循"

## 风险提示

1. **参数固定**：不要随意更改均线参数，否则信号不准确
2. **双重金叉**：单根均线信号不可靠，必须等待双重金叉
3. **回调等待**：金叉后不要立即买入，必须等待回调
4. **基本面检查**：技术分析不能替代基本面分析
5. **风险控制**：任何策略都有风险，必须设置止损，控制仓位

## 总结

135均线策略通过"三根线各管一摊"的核心思想，实现了从"凭感觉"到"有章可循"的转变。掌握这套战法，能够避开短线陷阱，稳稳拿捏大牛起涨点。

**记住**：
- 13线看资金活不活，向上站稳才有效
- 34线定趋势拐不拐，金叉不回才靠谱
- 55线守底线牢不牢，不破向上才稳妥
- 三根线配合好，大牛起涨信号藏不住

**对散户来说**：
- 这套135均线战法最大的好处就是简单、实用
- 不用天天盯盘熬通宵，每天花10分钟看看三根线的状态
- 就能精准找到大牛起涨点，不用追高、不用盲抄底，稳稳赚钱
